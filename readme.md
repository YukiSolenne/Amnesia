# AMNESIA · 失语记忆图书馆

（Amnesia Memory Library）

AMNESIA 是一个以 **“记忆即检索（Memory as Retrieval）”** 为核心理念的实验性项目，尝试将情感表达、信息组织与大语言模型结合，构建一个 **以“档案馆 / 图书馆”为隐喻的人机交互系统**。

在该系统中，AI 不被设定为“安慰者”或“心理咨询师”，而是作为 记忆的管理者，通过分类、检索、隐喻和结构化表达，回应用户提交的“记忆片段”。

本项目主要服务于课程设计与研究探索，同时也作为一个长期演进的个人项目原型。

此README.MD文档由AI辅助生成。

---

## 一、项目核心思想：**记忆即检索**

- 情绪被视作一种信息资源
- 人类与 AI 的理解路径相似，理解方式不同
- 通过显化情绪的结构与位置，构建新的陪伴与互动体验

在设计上，本项目受到以下概念启发：
- 信息组织
- 情绪结构化
- RAG、CoT

## 四、Emotion Schema（情绪结构）

项目采用一种 **结构化情绪表示方式**，通常包含以下字段：

```
{
  "summary": "中性概括用户表达内容",
  "keywords": ["关键词1", "关键词2"],
  "spectrum": {
    "valence": -1.0 ~ 1.0,
    "arousal": 0.0 ~ 1.0,
    "tones": ["情绪语调1", "情绪语调2"]
  },
  "thinking": "模型对情绪结构的内部推理",
  "metaphor_domain": "隐喻域",
  "metaphor_seed": 1024
}
```
